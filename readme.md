# NobleLife POS (NLI-POS)

NobleLife POS (NLI-POS) is a comprehensive **Point of Sale and management system** designed for NobleLife offices. It supports order processing, inventory management, multiple payment methods, reporting, and role-based access control, built using **ASP.NET Core Razor Pages**, **Entity Framework Core**, and **SQL Server**.

- Role-based access control (Admin, Cashier, Manager)  
- Audit logs for login, transactions, and changes  
- ASP.NET Core Identity for authentication and user management
